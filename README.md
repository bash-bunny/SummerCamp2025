## Kali

Instalacion en una [Kali Linux](https://www.kali.org/) las herramientas del curso.

```bash
# Descargar el script
git clone https://github.com/bash-bunny/SummerCamp2025.git
# Darle permisos
chmod u+x ./setup.sh
# Ejecutarlo
./setup.sh
```
