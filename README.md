## Kali

Instalacion en una [Kali Linux](https://www.kali.org/) las herramientas del curso.

```bash
# Descargar el script
wget https://raw.githubusercontent.com/bash-bunny/SummerCamp2025/refs/heads/main/setup.sh
# Darle permisos
chmod u+x ./setup.sh
# Ejecutarlo
./setup.sh
```
